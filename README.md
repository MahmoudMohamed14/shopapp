# shopapp

This is a  e-commerce application. I made by using api (postman), Shared Preference and State Management (Bloc)
## Tools
-api(postman)
- State management:
    - Bloc Design Pattern 

- Packages:
    - shared_preferences
    - conditional_builder_null_safety
    - flutter_bloc
    - bloc
    - carousel_slider
    -dio
